# 🚀 TrendTracker 발전 로드맵

TrendTracker의 사용자 경험 고도화와 시각적 완성도를 높이기 위한 지속적인 개선 계획입니다.

## 🎨 1. 디자인 및 시각적 에스테틱 (Visual Experience)

### 1.1 프리미엄 그라데이션 테마 및 레이아웃 안정화

- **그라데이션 배경**: 은은한 파스텔톤 그라데이션으로 고급스러운 분위기 조성.
- **레이아웃 안정성**: 여백 최적화를 통해 텍스트 겹침 현상을 완벽히 방지하고 스타일을 안정화.
- **텍스트 중심 카드**: 기존의 이미지를 배제하고, 뉴스 제목과 내용이 한눈에 파악되도록 텍스트 중심의 카드 디자인 적용.

### 1.2 미니멀리즘 가이드

- **아이콘 최소화**: 전문적인 느낌을 위해 불필요한 이모지 및 아이콘 삭제.
- **제목 시인성**: 굵고 큰 타이포그래피를 사용하여 핵심 내용을 즉각적으로 파악 가능하도록 구현.

## ✨ 2. 주요 기능 확장 (Features & UX)

### 2.1 검색 고도화 및 언어 모드

-키워드 검색할때 오타수정 기능 추가

- **언어 모드 설정**: 검색 전 한국어/영어 등 뉴스 검색 대상 언어를 선택할 수 있는 기능 추가.
- **코드 출력 지양**: 결과 분석 시 기술적인 코드 블록이나 불필요한 형식 배제.

### 2.2 사용자 경험(UX) 개선

- **뉴스 링크 및 날짜**: 각 카드에 뉴스 출처 링크와 발행 날짜를 명확하게 표시하여 신뢰성 확보.

## 🛠️ 3. 성능 최적화 (Technical Tasks)

### 3.1 로딩 및 검색 성능

- **성능 최적화**: API 호출 및 데이터 처리 로직을 개선하여 로딩 시간 단축.
- **스마트 렌더링**: 텍스트 중심의 가변 그리드 시스템 적용.
